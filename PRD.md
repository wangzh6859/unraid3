# Unraid-APP - Product Requirements Document

## 1. Vision
To provide Unraid users with a visually stunning, native-feeling mobile application that rivals modern commercial NAS systems (like fnOS) without sacrificing Unraid's inherent flexibility.

## 2. Core Modules
1. **Auth & Connection**
   - Save local IP and remote domain.
   - API key management for the Unraid API plugin.
2. **Dashboard**
   - Card-based UI showing array status (Started/Stopped/Parity Check).
   - Live resource graphs.
3. **Containers (Docker)**
   - List view with status indicators (Green dot for running).
   - Actions: Start, Stop, Restart, View WebUI, Logs.
4. **Virtual Machines**
   - List view with power states.
   - Actions: Force Stop, Restart.
