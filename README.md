# Unraid-APP 📱

A modern, fluid, and powerful mobile management application for Unraid servers, inspired by the clean aesthetics of fnOS.

## 🌟 Key Features

- **Dashboard**: Real-time CPU, RAM, Network, and Array status visualization.
- **Docker & VM Management**: Start, stop, restart, and monitor logs natively.
- **Storage Insights**: View disk health, SMART status, and capacity cleanly.
- **Multi-Server Support**: Manage multiple Unraid instances seamlessly.

## 🛠️ Technology Stack
- **Frontend**: Flutter / Dart
- **Backend API**: Unraid API (Community Plugin) GraphQL/REST integration.
