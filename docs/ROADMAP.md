## 功能优先级

### P0 - 必须
- 服务器添加/认证
- 基础状态监控
- Docker 容器管理
- 基础错误处理

### P1 - 重要
- 虚拟机管理
- 阵列管理
- 推送通知
- 多服务器支持

### P2 - 期望
- Widget 支持
- Wear OS / Apple Watch
- 快捷操作
- 深色模式自定义
- 多语言支持

## 技术债务追踪

| 项目 | 优先级 | 状态 | 备注 |
|------|--------|------|------|
| 单元测试 | High | Pending | Jest + React Testing Library |
| E2E 测试 | Medium | Pending | Detox |
| 性能监控 | Medium | Pending | Flipper / Sentry |
| 离线缓存 | Low | Pending | AsyncStorage + RTK Query |
